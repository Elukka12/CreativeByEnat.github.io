Portfolio Website

This is a repository for my personal portfolio website. It showcases my projects, skills, and experiences. 

## Features

- Home page with a brief introduction about me and my background.
- Portfolio section with a selection of my featured projects, including project descriptions, images, and links.
- Experience section highlighting my professional background and work history.
- Contact page with a form to get in touch and connect with me.
- Mobile-responsive design for optimal viewing across different devices.

## Technologies Used

- HTML5
- CSS3
- JavaScript
